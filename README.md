# 🏨 Hotel Booking Cancellation Prediction

## 📌 Project Overview
This project predicts whether a hotel booking will be canceled using Machine Learning.

## ⚙️ Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Random Forest Classifier

## 📊 Steps Performed
- Data Cleaning & Preprocessing
- Handling Missing Values
- Label Encoding
- Exploratory Data Analysis
- Model Building

## 🎯 Model Performance
- Accuracy: ~85% (approx)
- Evaluated using Precision, Recall, Confusion Matrix

## 📈 Key Insights
- Customers with long lead time are more likely to cancel
- Certain customer types show higher cancellation rates

## 🚀 How to Run
1. Install required libraries
2. Run `model.py`
